# Job Application Tracker

A simple, minimal web app to track job applications. Built with vanilla HTML, CSS, and JavaScript. No backend requiredâ€”data is stored locally in your browser.

## Features
- Add, edit, and delete job applications.
- Search and filter by company, position, or status.
- Dashboard with basic statistics.
- Export applications to CSV.
- Responsive design.

## How to Use
1. Clone or download this repo.
2. Open `index.html` in your browser.
3. Start adding applications!

## Technologies
- HTML5
- CSS3
- JavaScript (ES6)
